# The AI Apprentice Website

Static launch site for The AI Apprentice.

## Pages

- `index.html` - homepage
- `course.html` - six-week course outline
- `signup.html` - signup page
- `about.html` - about Roger and Jane
- `resources.html` - starter prompts and checks
- `advanced.html` - next milestones / advanced track
- `contact.html` - contact page

## Deploy On Render

Use a Static Site.

Settings:

```text
Name: ai-apprentice-site
Build Command: leave blank
Publish Directory: .
```

No server is required.

## Before Public Launch

- Connect `signup.html` to an email capture tool.
- Confirm public contact email.
- Connect the custom domain.
